Nest Backend

Backend server built within nestjs and node.

This server manages users and its private notes, using json web tokens the server can authenticate and verify users.


Thanks to authentication, notes created by certain user can only be accessed (CRUD) by its owner.


The security its highly important for this application, so, the are a lot of unit test implemented along all the project to ensure that all is working correctly.


In order to test the interaction of the application with the database, for those components (services) wich interact directly with the database, a fake and test only database is created for unit testing using better-sqlite3.


This way, the unit test ensures that all relation ships and entities columns metadata are great and that the aplication will not crash in production. This extra security could not be access by just mocking the database.
